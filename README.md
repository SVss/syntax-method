## Ограничения

1. Тип терминала сразу указан
1. Строгий порядок правил (от конкретных к более высокоуровневым)
1. Упорядоченный список входных терминалов

## Грамматика

### Терминалы:
- вертикаль: |
- горизонталь: -
- диагональ влево: \
- диагональ вправо: /

### Таблица переходов:
1. / \ → крыша
1. | | → стены
1. стена - → основание |_|
1. основание - → корпус
1. корпус крыша → дом

## Пример:
1. / \ - | | -
1. крыша - | | -
1. крыша - стены -
1. крыша основание -
1. крыша корпус
1. дом
